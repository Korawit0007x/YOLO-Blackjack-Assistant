# YOLO Blackjack Assistant

A real-time Blackjack assistant using YOLO and OpenCV.

## Features

* Detect playing cards from webcam
* Calculate Blackjack score automatically
* Show detected cards in real time
* Display BLACKJACK when score equals 21

## Technologies

* Python
* OpenCV
* Ultralytics YOLO
* Computer Vision
